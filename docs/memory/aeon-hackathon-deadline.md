ÆON Home is being built for the Snapdragon Multiverse Hackathon at Qualcomm Noida.
Hacking started 2026-07-18 1:00 PM IST; **submission deadline is 2026-07-19 1:00 PM IST**,
demos 1–4 PM, winners ~5 PM. Demo slot is **5 minutes, hard-timed**.

Hard submission requirements: public GitHub repo, README with team names + emails,
setup-from-scratch and run instructions, an open-source license, and the app must be
**runnable from those instructions**. Majority of compute must run on the edge.

Judging: Technical Implementation 40 / Use-Case & Innovation 25 / Deployment &
Accessibility 20 / Presentation & Docs 15. A separate 100-point prize for Multi-Device
Orchestration Excellence — that one is the natural fit for this project's phone → node →
leaf + PC fan-out.

**Why:** time pressure is the dominant constraint; a working end-to-end demo beats a more
complete but unrunnable system, and "ease of installation" is 20% of the score.

**How to apply:** prefer zero-build-step, single-`pip install` choices; keep the failure
demos (PC offline, leaf offline, unsigned command) working since they carry the
orchestration prize. See [[aeon-phase-plan]].
